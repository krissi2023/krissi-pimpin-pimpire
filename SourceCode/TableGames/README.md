# Table Games

This directory contains classic table games for the Krissi Pimpin' Pimpire gaming platform.

## Available Games
- **Draw** - A card drawing game
- **Blackjack** - The classic casino card game (21)
- **High or Low** - Card prediction game

## Game Types
- Classic casino table games
- Card-based table games
- Dealer-operated games

## Development
Add new table games as separate modules in this directory.
